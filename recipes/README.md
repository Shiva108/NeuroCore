behavior.

  blessed local profile
- [`hosted-stack-quickstart`](./hosted-stack-quickstart/README.md) for the
  blessed hosted profile
- [`slack-slash-flow-report`](./slack-slash-flow-report/README.md) for Slack
  slash-style review reporting
- [`discord-slash-flow-report`](./discord-slash-flow-report/README.md) for
  Discord slash-style review reporting
- [`security-memory-review-report`](./security-memory-review-report/README.md)
  for opinionated security review reports
- [`ops-weekly-memory-report`](./ops-weekly-memory-report/README.md) for
  opinionated weekly ops reporting
